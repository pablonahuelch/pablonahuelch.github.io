SERVICIOS SANTA MARIA – Registro de Pozos

Portal simple para registrar clientes, pozos e históricos de manera progresiva e intuitiva.

Alta

Registrar cliente

Desde este bloque podés dar de alta un cliente nuevo. El sistema asigna el ID automáticamente y guarda la base para luego registrar pozos y sus históricos.

Consulta y gestión

Consultar clientes y gestionar pozos

Desde este bloque podés buscar un cliente ya cargado, ver sus pozos, registrar uno nuevo, agregar históricos o editar registros existentes.

Alta de cliente

Registrar nuevo cliente

Complete solo los datos necesarios. El ID se genera automáticamente.

Gestión de clientes, pozos e históricos

1. Seleccionar cliente

Primero elegí un cliente ya cargado para trabajar sobre sus pozos.

Cliente seleccionado

Desde aquí puede avanzar con las acciones disponibles para este cliente.

Registrar nuevo pozo

El pozo quedará vinculado al cliente seleccionado. El ID del pozo se genera automáticamente.

Pozos del cliente

Seleccione un pozo para consultar sus históricos o cargar uno nuevo.

Detalle del pozo

Desde este bloque puede trabajar el histórico del pozo seleccionado.

Agregar histórico

El histórico se vincula automáticamente al pozo seleccionado.

Editar histórico existente

Seleccione un registro histórico del pozo para modificarlo.

Históricos del pozo

Listado completo de históricos asociados al pozo seleccionado.
